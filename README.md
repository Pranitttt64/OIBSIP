# 🚀 Projects Completed

## 1️⃣ Iris Flower Classification 🌸

### 📌 Objective
Predict the species of iris flowers using supervised classification techniques.

### 🔍 Techniques Used
- Logistic Regression
- K-Nearest Neighbors (KNN)
- Support Vector Machine (SVM)
- Feature Scaling
- Confusion Matrix Analysis

### 📊 Outcome
Achieved high accuracy in predicting:
- Iris-setosa
- Iris-versicolor
- Iris-virginica

---

## 2️⃣ Unemployment Analysis in India 📉

### 📌 Objective
Analyze unemployment trends across Indian states and study the impact of COVID-19.

### 🔍 Analysis Performed
- Data Cleaning & Preprocessing
- Time Series Trend Analysis
- State-wise Comparison
- Rural vs Urban Comparison
- Heatmap Visualization
- COVID-19 Impact Study

### 📈 Key Insights
- Significant unemployment spike during 2020 lockdown.
- Urban areas experienced higher volatility.
- Certain states consistently reported higher unemployment rates.

---

## 3️⃣ Car Price Prediction 🚗

### 📌 Objective
Build a regression model to predict car selling prices based on multiple features.

### 🔍 Techniques Used
- Feature Engineering (Car Age)
- Categorical Encoding
- Linear Regression
- Random Forest Regressor
- Pipeline Implementation
- Model Evaluation (R², MAE, RMSE)
- Feature Importance Analysis

### 📊 Result
Random Forest outperformed Linear Regression and provided better predictive accuracy.
